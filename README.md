The target audience is NBA fans who want to search NBA player stats. Although current NBA website allows the fan to search player stats, the website does not work intuitively. For example, the website cannot return multiple players, need to resolve name perfectly, and the user interface is hard to see. Therefore, I decided to implement a new NBA website.
